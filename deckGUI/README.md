HOW TO USE:



1. ONCE THE FILE IS DOWNLOADED, GO INTO COMMAND PROMPT, AND CHANGE TO THE DIRECTORY OF WHICH YOUR GUI\_CUSTOMIZATION.PY FILE IS IN



2\. RUN THE FILE BY DOING "Python gui\_customization.py", IT WILL OPEN A TAB (MAKE SURE TO HAVE ALL THE REQUIRED PACKAGES DOWNLOADED FIRST)



3\. PRESS THE BOX THAT YOU WANT TO HAVE THE LINK SET TO, AND COPY AND PASTE THE WANTED LINK IN THE TEXT BOX AND PRESS SAVE



4\. NOW CLOSE THAT TAB, AND RUN THE "gui.py" FILE, THIS WILL BE IN A DIFFERENT DIRECTORY, NOW WHEN YOU PRESS THE SAME BOX, IT WILL OPEN THAT FILE



(IF YOU WANT IMAGES SET TO THAT BUTTON, JUST CHANGE THE FILE PATH IN THE "gui.py" TO THE PATH OF WHICH YOUR IMAGE IS AT)
